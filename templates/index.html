OpenMensa Parser Bremen

This is a parser for every Mensa in Bremen that is managed by the Studirenden Werk Bremen.

It is written for the OpenMensa project and therefore returns an XML Feed corresponding to the OpenMensa feed schema.

To use this parser just append the name of the mensa to the url and the retuned page consits of the XML feed for the mensa.

* Uni-Mensa: "/mensa/uni-mensa"
* Cafeteria GW2: "/cafeteria/gw2"
* Mensa Neustadtwall: "/mensa/neustadtswall"
* Mensa Werderstraße: "/mensa/werderstraße"
* Mensa Flughafen: "/mensa/airport"
* Mensa Bremerhaven: "/mensa/bremerhaven"
* Cafeteria Bremerhaven: "/cafeteria/bremerhaven"
* Interimsmensa HFK: "/mensa/interimsmensa-hfk"
